#학습

#작업 생성

  • 지금까지 작업한 데이터를 가지고, 이미지 학습을 진행하기 위해 작업 화면으로 이동한다.
  • 우측 상단의 분석 탭에서 서비스 메뉴 사이를 이동할 수 있다.
  • 여기에서 작업을 선택하면 작업 관리 화면으로 넘어간다.

  • 작업 생성 버튼을 눌러 작업을 생성한다.

  • 작업 생성 방식을 선택하는 팝업이다.
  • DL Modeler에 내장된 image classification 알고리즘으로 학습을 진행할 예정이므로 생성 방식으로 Basic을 선택한다.

  • 그 다음 나오는 작업 생성(Basic) 팝업에서는 작업 상세 정보를 입력한다.
  • 먼저, 작업 명과 작업 설명을 입력한다.
    • 작업 명: image-classification
    • 작업 설명: image classification demo

  • 알고리즘은 이미지 분류를 선택한다.
  • 내장 image classification 알고리즘은 tensorflow 기반으로 만들어졌으며, 학습 신경망 layer는 6개로 구성된다.
  • 알고리즘을 선택하면 그에 따라 데이터를 선택하는 부분이 바뀐다. 위 칸에는 학습 이미지, 아래 칸에는 학습 label 경로를 입력한다.
  • 오른쪽에 있는 폴더 아이콘을 각각 눌러 파일 경로를 찾는다.

  • 이미지 경로를 찾을 땐 train-image '폴더'를, 레이블 경로를 찾을 땐 train-label '폴더'를 선택 후, 확인을 클릭한다.
    • 이미지 경로 mnt > project > image-classification_324_edu01 (사용자에 따라 다름) > data > train-image > train-image (폴더 선택)
    • 레이블 경로 mnt > project > image-classification_324_edu01 (사용자에 따라 다름) > data > train-label > train-label (폴더 선택)
  • 데이터는 데이터 탭에서 업로드해 두었던 학습 이미지와 label csv 파일이다.
  • 모든 정보를 입력하면 생성 버튼이 활성화되는데, 생성 버튼을 눌러 작업을 생성한다.

#학습생성

  • 작업 관리 화면에 생성한 작업이 보이게 된다. 작업명을 클릭하면 학습 관리 화면이 나타난다.

  • 학습 관리 화면에서 학습을 생성하고 관리할 수 있다. 학습 생성 버튼을 눌러 학습 생성을 시작한다.

  • 다음은 학습을 생성하는 과정이다.

a. 학습에 사용할 인스턴스(cpu/gpu)를 선택한다. gpu를 선택한 경우 gpu 개수도 같이 입력한다.

b. 하이퍼파라미터 중, num_types는 6으로 설정해준다. demo 이미지의 카테고리 수가 6개이기 때문이다. 다른 하이퍼파라미터는 원하는 값으로 입력한다. (기본 값으로 두어도 무방하다.)

종류값
batch_size100
epochs100
learning_rate0.001
num_types6
filter_num120
filter_num240
filter_num360
filter_num480
filter_num5100
filter_num6100

c. 학습명 입력 칸에 학습명을 입력한다.

- 학습명: image-train-cpu

d. 생성 버튼을 누르면 이미지 학습이 시작된다.

  • 학습 추가 버튼을 누르면 학습명과 하이퍼파라미터를 입력하는 칸이 늘어나며, 한 화면에서 여러 개의 학습을 동시에 생성할 수 있다. 단, 프로젝트에 할당 된 자원을 초과하면 안 된다.

#학습진행 확인

  • 학습 관리 화면에 생성한 학습이 보인다.
  • 학습 상태는 학습 대기/학습 중/학습 완료/학습 실패로 표시된다.
  • 상태는 자동 새로 고침이 되지 않기 때문에 사용자가 직접 새로 고침 버튼을 눌러 주기적으로 확인해야 한다.
  • 생성된 학습 명을 클릭하여 학습 상세 페이지로 진입한다.

  • 학습 상세 팝업의 결과 그래프 탭에서 학습 진행 상황을 한 눈에 볼 수 있다.
  • 학습 실패/학습 완료 시 자세히 보기 버튼이 활성화되며, 이를 누르면 해당 학습에서 나오는 로그를 확인할 수 있다.
  • 배포 버튼은 학습 완료 후 모델이 생성되면 활성화된다.
  • 학습 진행 시간은 하이퍼파라미터 설정, 데이터의 양, 학습 인스턴스에 따라 차이가 난다.
    • 기본 값 하이퍼파라미터, 59개 학습 데이터 기준으로 CPU 학습은 약 17분, GPU 학습은 약 7분 소요.
    • 기본 값 하이퍼파라미터, 14,034개 전체 학습 데이터로 모델을 만들면 CPU 학습 약 15시간, GPU 학습 약 17분 소요.
