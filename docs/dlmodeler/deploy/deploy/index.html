Deploy

배포#

DL Modeler의 배포 기능에서는 작업에서 학습을 마친 모델을 기반으로 배포 기능을 제공하며 배포된 서버를 관리할 수 있다.

배포관리#

  • 배포 관리는 작업상세의 학습 목록에서 배포한 배포 서비스를 관리한다.
  • 해당 프로젝트에서 배포된 서버들의 목록과 서버 기본 정보를 확인할 수 있다.
  • 액션의 재생버튼, 정지버튼, 휴지통버튼을 통해 활성화, 비활성화, 삭제가 가능하다. 삭제의 경우, 배포 정지 후에 가능하다.
  • Active 상태인 서비스는 배포서버 IP를 통해 rest api 호출이 가능하다.

배포상세#

  • 배포 관리에서 특정 배포 서버를 클릭하면 배포 상세 화면이 나타난다.
  • 예측 관리 화면에서 UI로 진행한 예측 목록과 결과를 확인할 수 있다.
  • 예측생성버튼으로 데이터셋에 등록 된 데이터를 이용해 새로운 예측을 만들 수 있다.

  • 상세 탭에서 배포서버와 모델, input 입력 정보 등을 볼 수 있으며 API호출 시 예측 예시를 활용하여, 터미널에서 curl 명령어를 활용하여 REST API예측을 할 수 있다.
  • 배포서버 설명과 최소 서버 수, 최대 서버 수를 수정할 수 있다. 단, 최소 서버 수는 줄일 수 없는데, 기존에 배포된 서비스를 내리는 과정에서 API 서비스 장애를 발생시킬 가능성이 있기 때문이다.

예측생성#

예측 관리 화면에서 ‘예측생성’ 버튼 클릭 시 예측 생성 레이어팝업이 노출된다. 본 기능에서는 선택한 테스트 데이터의 예시를 볼 수 있으며, 샘플 예측을 통해 예시 데이터의 샘플링 결과를 조회할 수 있다.

  • 예측생성 버튼을 누르면 예측 생성 화면이 나타난다.
  • 값을 입력하고 경로검색버튼을 클릭해 데이터세트에서 등록한 예측대상 데이터를 불러온다. 폴더를 선택하여 폴더 내 데이터를 한꺼번에 가져올 수도 있다.
  • 예시 리프레시를 클릭하면 특정 데이터 1개가 나타난다.
  • 샘플예측을 클릭해 예시데이터의 결과를 예측한다.
  • 샘플예측까지 수행하면 생성이 가능하다.

예측상세#

  • 예측 작업의 진행률 및 모델의 정보와 배포 정보, 예측 결과를 확인할 수 있다.
