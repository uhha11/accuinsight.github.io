Model

Model#

저장된 모델의 조회, 수정, 삭제 및 Prediction 할 수 있다.

Model 조회#

저장된 모델을 조회한다.

  • 이름이나 기간을 이용하여 Model을 검색한다.
  • Model을 클릭하여 Pipeline란에서 Data 처리 흐름을 볼 수 있다.
  • Edit(수정) 버튼을 클릭하여 Model을 수정한다.
  • Remove(삭제) 버튼을 클릭하여 Model을 삭제한다.
  • Predict 버튼을 클릭하여 Model을 이용한 Prediction(예측)한다.

Model수정#

저장된 Model정보를 수정한다.

  • Model을 선택한 뒤, 우측 상단 Edit(수정) 버튼을 클릭하면 수정창이 나타난다.
  • Model의 이름과 설명을 수정한다.
  • Save 버튼을 눌러 수정사항을 저장한다.
  • 닫기 버튼을 눌러 수정사항을 취소한다.

Model 삭제/Import및Export#

  • [삭제] 선택 후 상단의 삭제버튼을 클릭하여 Model을 삭제한다.
  • [Import/Export] 각 Model 테이블 우측 다운로드 아이콘을 클릭하여 다운로드 한다. 우측 상단 Import 버튼을 클릭하여 다운받은 zip 파일 또는 hdfs, s3, icos에 저장되어 있는 모델을 import 한다.

Prediction#

Model을 이용하여 새로운 데이터를 Prediction한다. 모델을 하나 선택 하고 우측 상단 Predict 버튼을 클릭하여 창을 연다

  • Select Dataset 옆의 폴더버튼을 클릭하여 테스트 데이터를 가져온다.
    • 목록에서 Model에 적용할 Dataset을 선택한다.
    • 선택 시, “Predict” 창에서 보여지는 학습데이터의 Schema와 “Load TestData” 창에서 보여지는 테스트 데이터의 Schema를 비교하여 가능한 Dataset인지를 확인해야한다.
  • Predict버튼을 클릭하여 선택된 Test Dataset에 모델을 적용하여 예측한다.

  • Prediction 결과를 저장할 Dataset 정보를 입력한다.
  • 예측된 결과는 입력한 저장 경로에 저장되며, DATASET 화면을 통해서도 확인 가능하다.
