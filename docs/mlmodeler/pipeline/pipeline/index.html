#Pipeline(파이프라인) 조회

사용자가 저장한 Pipleline의 목록과 Pipeline의 상세 내용을 조회한다.

  • Pipeline 이름이나 기간을 이용하여 Pipeline을 검색한다.
  • 조회된 목록 중 하나의 Pipeline을 선택하면 Data 처리 흐름을 볼 수 있다.
  • Pipeline 화면의 알고리즘을 선택하면 학습한 알고리즘의 Property(프로퍼티)를 확인할 수 있다.

#Pipeline 열기/삭제/Import및Export

  • [Open] Pipeline을 선택한 뒤, 우측 상단 Open(열기) 버튼을 클릭하여 선택한 Pipeline을 Workspace (워크스페이스) 화면에 연다.
  • [Remove] Pipeline을 선택한 뒤, 우측 상단 삭제 버튼을 클릭하여 선택한 Pipeline을 삭제한다.
  • [Import/Export] 각 Pipeline의 테이블 우측 다운로드 아이콘을 클릭하여 다운로드 한다. 우측 상단 Import 버튼을 클릭하여 다운받은 json 파일을 import 한다.
