DHP

DHP 개요 (Dynamic Hadoop Provisioning)#

소개#

DHP는 On-demand 방식의 간단한 APIs 호출 방식으로 최적화된 컨테이너 기반 하둡 클러스터 제공합니다.

  • 클라우드 상에서 데이터 분석에 필요한 빅데이터 프레임워크인 Apache 하둡을 간편하게 실행시킬 수 있는 서비스를 제공
  • On-premise/ IBM Cloud 환경에서 하둡 구성을 위한 별도의 리소스 생성 없이, 제공되는 image 기반 하둡 클러스터를 자유롭게 생성/삭제하고 이를 통해 보다 빠르게 비지니스 분석 업무를 처리할 수 있는 기회

특장점#

1. 신속성#

별도의 인프라 구축 없이 하둡 클러스터를 신속하게 구성하고, 개발자는 데이터 분석을 위한 BIZ 요건 개발에만 집중할 수 있습니다.

2. 비용 효율적#

하둡 플랫폼을 도입하는데 있어 소모되는 비용 및 기술적 문제들을 클라우드 상에서 손쉽게 구성함으로써 CAPAX(설비투자) / OPEX(운영비용) 측면에서 비용을 절감할 수 있습니다.

3. 다양한 하둡 에코 시스템 제공#

데이터 처리를 위해 다양한 하둡 에코 시스템 (Oozie / Hive / Spark) 을 빠르게 생성 및 회수 할 수 있습니다.
