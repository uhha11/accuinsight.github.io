DCM (Dynamic Cluster Manager)#

DCM 소개#

On-Premise/Public Cloud의 다양한 환경에서 Cluster를 손쉽게 배포 및 관리하여 자원의 효율적인 활용을 지원하는 서비스입니다.

  • MCM (for Public Cloud) : Cloud Service Provider 환경에 맞춰 AWS의 EMR, GCP의 Dataproc Cluster 생성 및 관리합니다.

  • DHP (for On-Premise/Private Cloud) : 인프라/분석 SW 관리 최적화 관리형 Hadoop / Spark 클러스터 자동 배포 서비스를 제공합니다.
