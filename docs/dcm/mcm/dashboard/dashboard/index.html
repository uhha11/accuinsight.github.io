#Main 화면

  1. Cluster
  • 동작하고 있는 클러스터를 클라우드 서비스 별로 표시
  • 클러스터 상태 값에 따라서 Healthy와 Unhealthy로 구분
    • Healthy - WAITING, RUNNING, STARTING, BOOTSTRAPPING, CREATING, UPDATING
    • Unhealthy - ERROR
  • 다음 클러스터 상태는 포함 되지 않음
    • DELETING, TERMINATING, TERMINATED, TERMINATED_WITH_ERRORS, UNKNOWN
  1. Nodes
  • 클러스터를 구성한 노드를 클라우드 서비스 별로 표시
  • 노드의 상태 값에 따라서 Active와 Inactive로 구분
    • Active - PROVISIONING, STAGING, PROVISIONING, BOOTSTRAPPING, RECONFIGURING, RESIZING, RUNNING
    • Inactive - STOPPING, STOPPED, SUSPENDING, SUSPENDED, TERMINATING, TERMINATED, ARRESTED, SHUTTING_DOWN, ENDED
  • 다음 클러스터 상태는 포함 되지 않음
    • DELETING, TERMINATING, TERMINATED, TERMINATED_WITH_ERRORS, UNKNOWN
