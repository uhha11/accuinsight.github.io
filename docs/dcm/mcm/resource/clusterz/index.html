클러스터 관리

#AWS 클러스터 관리

① 필터조회 영역

검색필터 버튼을 클릭하면 입력창이 활성화 됩니다. 상태, 이름, 생성 기간, 생성자로 조회 가능합니다. 입력된 문자가 포함된 데이터 혹은 입력된 기간에 생성된 데이터만 로드되어 화면에 표시됩니다.

② 클러스터 생성

우측상단 클러스터 신청 버튼을 클릭하면 클러스터 생성 팝업이 활성화 됩니다.

필수 입력사항

  • 이름
  • 템플릿 닉네임(선택)
  • 인스턴스 수(Master, Core, Task)
  • 릴리즈 버전(선택)

릴리즈 버전을 클릭하면 기본적으로 설치되는 어플리케이션을 확인할 수 있습니다. 타입을 Custom으로 변경하면 추가로 설치할 수 있는 어플리케이션 리스트가 표기되어 선택된 항목이 설치됩니다. 필수 입력사항 입력후 생성버튼을 클릭하면 클러스터가 생성됩니다.

③ 노드변경 버튼

클러스터의 상태가 TERMINATING 또는 TERMINATED 상태가 아닐경우 노드변경 아이콘을 클릭하면 노드변경 팝업이 나타납니다.변경할 노드수를 입력후 변경 버튼을 클릭하면 노드 수의 변경이 가능합니다. 최대 사용가능 노드 수는 클러스터 생성때 선택한 템플릿의 최대노드수 입니다

④ 클러스터 종료

종료할 클러스터들을 체크 후 우측상단 종료버튼을 클릭하면 종료확인 팝업이 나타나고 확인 후 일괄종료 됩니다. 종료할 클러스터의 Action아이콘 중 종료 아이콘을 클릭하면 종료확인 팝업이 나타나고 확인 후 종료가 가능합니다.

⑤ 스케줄 설정

스케줄을 등록하면 스케줄러가 예약 일시에 맞춰서 자동으로 클러스터 시작/중지를 수행 합니다.
