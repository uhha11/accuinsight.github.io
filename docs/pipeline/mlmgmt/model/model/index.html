#Model

#Model 조회

저장된 모델을 조회 및 관리합니다.

  1. 검색 : 모델 이름, 기간을 이용하여 모델을 검색
  2. 불러오기 : 모델을 신규 등
  3. 수정 : 모델 명, 설명 수정
  4. 삭제 : 모델 삭제
  5. 파이프라인 : 모델을 클릭하여 Pipeline란에서 Data 처리 흐름 확인
  6. Action icon
    • Results : 해당 모델명 또는 Results 버튼을 클릭하여 모델 수행 결과 확인
    • 다운로드 : 해당 모델을 로컬로 다운로드

#Model 불러오기

  1. 로컬에 zip 파일 또는 hdfs, s3, icos에 저장되어 있는 모델을 선택
  2. Name : Model 명 입력
  3. Description : Model 설명 입력
  4. 로컬에 zip 파일을 선택한 경우, 업로드 위치 선택

#Model 수정

저장된 Model의 이름과 설명을 수정 합니다.

#Prediction

모델 수행 결로 Training Result, Model Description, Spark ML code를 확인 할 수 있습니다.
