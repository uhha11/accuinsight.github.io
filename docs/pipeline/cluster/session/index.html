#세션관리 개요

워크플로우 생성/수정 시 데이터 스냅샷, 스키마 조회, ml 컬럼가져오기 기능을 위한 spark livy 세션을 관리합니다.

  1. 검색필터 : 버튼 클릭하여 필터창 활성화 후 검색대상 클러스터명 입력/검색
  2. 재검색 : 세션 리스트 재조회
  3. 클러스터 목록 및 spark livy 세션 조회
    • ID : 클러스터 ID
    • 클러스터명 : 클러스터 이름
    • 클러스터 타입 : dhp, on-premise의 클러스터 구분
    • 정보 : 클러스터 생성시 입력한 설정 정보 간략 출력 ('…' 아이콘 마우스오버시 간추린 정보 확인가능)
    • 세션ID : spark livy의 세션 ID
    • 상태 : 세션의 상태 busy(작업중), idle(정상 대기 상태), starting(시작중), stop(정지), unknown(확인불가)
    • 소유자 : 클러스터 생성자
    • 기능 : 순서대로 [세션생성], [세션삭제], [로그조회], [결과조회], [코드실행]을 담당

#세션관리 기능 소개

#로그조회

spark livy 세션 로그를 조회합니다.

#결과 조회

수행된 spark 코드 및 수행결과를 조회합니다. 이전에 실행한 이력도 조회 가능하며 실패 시 에러 로그도 확인할 수 있습니다.

#코드 실행

코드 실행창을 열어 직접 코드작성 및 실행이 가능합니다.
