cluster

#Definition

[데이터불러오기]에서 불러온 데이터를 클러스터로 나누고, 랜덤하게 선택된 특정 클러스터의 전체 데이터를 샘플링할 때 사용합니다.
좌측 [데이터샘플링]노드 중 [cluster]노드를 drag & drop 한 후 Property 항목을 입력합니다.

#Set

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

#Property

[Node Description] 작성 중인 노드명 입력

  1. fraction : 리턴할 데이터셋과 전체 데이터셋 간의 크기 비율을 의미하며, 0~1사이의 값 입력
  2. col : 적용할 기준 컬럼 선택
  3. includeJob : 배치/스케줄링 작업 수행 시 샘플링 포함여부

#Example

random 샘플링에서 사용한 HR_Retention.csv(14,999 records)파일에서 cluster 샘플링작업을 수행합니다. 기준컬럼(col)을 sales로, fraction을 0.1로 설정할 경우 sales 컬럼에 포함된 10개 category 중 10%만 클러스터를 만들고 해당 클러스터의 전체 데이터를 샘플링합니다.

  1. [HDFS불러오기], [cluster] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [cluster]노드의 fraction 값을 0.1로 선택하고, col값을 sales로 선택
  3. sales cagegory(10개) 중 랜덤하게 1개를 선택한 후 포함된 전체데이터를 샘플링
