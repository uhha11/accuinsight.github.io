distinct

Definition#

중복되는 행을 삭제합니다.
좌측 [데이터처리(기본)]노드 중 [distinct]노드를 drag & drop 합니다. [distinct]노드는 입력할 Property가 없습니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

Example#

상품정보에 포함된 중복데이터를 제거합니다.

  1. [HDFS불러오기], [select], [distinct]노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [select]노드 snapshot선택시 조회되는 중복데이터가 [distinct] 노드 적용후 제거됨
