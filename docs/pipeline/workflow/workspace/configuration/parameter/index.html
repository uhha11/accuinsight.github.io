Parameter

#Parameter

Parameter 패널을 클릭하면 하단 패널이 확장된다.

  1. parameters : key-value 형태로 워크플로우에서 파라미터로 넘길 변수를 입력한다.

    ${변수명}으로 노드별 프로퍼티에서 활용

  2. globalVariable : 변수관리를 통해 등록된 변수 그룹을 선택한다.

    parameters 보다 우선순위가 높음
