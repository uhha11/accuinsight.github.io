python

Definition#

python 코드를 작성하여 실행하는 노드입니다. property 패널 "8.source"에 python 코드를 입력합니다. Source Editor 버튼("8.source" 우측 버튼) 클릭 시 Editor가 팝업되며 보다 큰 화면에서 코드를 작성할 수 있습니다.
좌측 [Flow구성]노드 중 [python]노드를 drag & drop 한 후 Property 항목을 입력합니다. Property 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 Property 항목을 볼 수 있습니다.

Set#

[setting], [scheduler], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

property#

[Node Description] 작성 중인 노드명 입력

property 중 2~6 항목은 [연결관리]에 등록된 원격서버 정보 선택 시 자동으로 입력되며, 원하는 서버목록이 없을 경우 [설정] > [원격 서버]에서 필요서버를 먼저 등록합니다.

  1. appName : appName 입력
  2. Python 실행 IP : [설정] > [원격 서버]에 저장된 서버 IP, PORT
  3. user : [설정] > [원격 서버]에 저장된 서버 사용자 아이디
  4. password : [설정] > [원격 서버]에 저장된 서버 사용자 패스워드
  5. 원격서버ID : [설정] > [원격 서버]에 저장된 서버 ID
  6. 원격서버명 : [설정] > [원격 서버]에 저장된 서버명
  7. sourcepath : 브라우저를 통해 원격서버에 저장된 소스선택 후 불러오기 기능
  8. source : 실행할 python 코드 입력
    • [Source Editor]를 선택해서 팝업된 editor를 통해 코딩 가능
  9. argument : 인수 설정
  10. forceOK : 실패 시 강제 OK 처리 여부

Example#

원격서버에 저장된 csv 파일을 읽어 txt 파일로 생성하는 파이선 코드 예제입니다. 수집서버로 전달되는 원천 데이터를 python으로 전처리하는 업무가 있을 경우 적용 가능합니다.

  1. hdfsToServer 노드를 사용해서 원격지 서버로 csv파일 이동

    • /home/dpcore/AIR_HOUR_2019_3.csv

  2. 연결관리에서 원격서버를 선택

  3. [8.source] 옵션에 아래와 같이 입력 

    import csv, sys
sys.stdout = open('output_air_hour2019.txt','w') 
with open('/home/dpcore/AIR_HOUR_2019_3.csv','r') as file:    reader  = csv.reader(file)        for record in reader:        print(record)

  1. 저장 및 실행 후 원격서버에 output_air_hour2019.txt 파일 생성 확인
