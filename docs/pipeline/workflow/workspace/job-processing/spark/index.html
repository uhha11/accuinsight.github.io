spark

#Definition

작성한 코드를 Spark에서 실행하는 노드이다. [Flow구성]노드 중 [spark]노드를 drag & drop 한 후 Property 항목을 입력한다. Property 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 Property 항목을 볼 수 있다.

#Set

[setting], [scheduler], [parameter] 설정은 [워크플로우 생성] > [설정]을 참고한다.

#property

[Node Description] 작성 중인 노드명 입력

  1. prepare : 노드 실행 결과를 HDFS에 저장하기 위해 파일삭제/폴더생성 작업 선 수행(워크플로우 반복실행 시 활용가능)

    • delete : 노드 실행 전 삭제할 폴더/파일 경로
    • mkdir : 노드 실행 전 생성할 폴더 경로

  2. file : 노드에서 사용할 라이브러리 경로를 설정

  3. archive : archive경로를 설정

  4. retry

    • max : 실행 실패 시 재시도 횟수
    • period : 재시도 주기(분 단위)

  5. property : 실행에 사용할 property (key, value) 입력

  6. master : 사용할 spark url 입력(DHP에서 생성한 클러스터 사용시 yarn 입력)

  7. mode : spark 실행 모드(client, cluster) 입력(DHP에서 생성한 클러스터 사용시 client 입력)

    deploy-mode설명
    clientdriver program을 로컬에서 실행
    clusterdriver program을 워커 모신 중 하나에서 실행

  8. name : spark 애플리케이션명

  9. jar : 실행할 python(py), jar 등의 경로 입력

  10. sparkOpts : spark 실행시 사용할 옵션정보 입력(key, value)

    keyvalues설명
    --driver-cores1spark driver에서 사용할 core 수 지정
    --driver-memory1024mspark driver에서 사용할 memory 지정
    --executor-cores1executor에서 사용할 core 수 지정
    --executor-memory1gexecutor에서 사용할 memory 지정
    --num-executors1실행할 executor 수 지정
    --queuedefaultjob을 제출할 yarn queue 이름 지정
    --confPROP=VALUE임의의 spark 환경설정 속성값 지정

[Note] 아래 내용참고
https://spark.apache.org/docs/latest/running-on-yarn.html

  1. argument : python code 실행시 전달할 argument
  2. jobXml : jobXml 경로 입력. 잡에 전달할 프로퍼티를 별도 xml 로 작성하여 전달가능
  3. forceOK : 데이터 처리가 실패해도 정상으로 표시하고 종료

#Example

입력받은 숫자의 평균을 구하는 basicavg.py 파일을 spark에서 실행하는 예제이다. 

"""
>>> from pyspark.context import SparkContext
>>> sc = SparkContext('local', 'test')
>>> b = sc.parallelize([1, 2, 3, 4])
>>> basicAvg(b)
2.5
"""
import sys
from pyspark import SparkContext
def basicAvg(nums):
    """Compute the avg"""
    sumCount = nums.map(lambda x: (x, 1)).fold(
        (0, 0), (lambda x, y: (x[0] + y[0], x[1] + y[1])))
    return sumCount[0] / float(sumCount[1])
if __name__ == "__main__":
    master = "local"
    if len(sys.argv) == 2:
        master = sys.argv[1]
    sc = SparkContext(master, "Sum")
    nums = sc.parallelize([1, 2, 3, 4])
    avg = basicAvg(nums)
    print avg

  1. 워크플로우 실행 클러스터에 basicavg.py 파일을 적재한다.

  2. [Flow구성] > [spark] 노드를 drag & drop 한 후 setting 패널에서 실행클러스터를 선택한다.

  3. property 패널에서 아래와 같이 입력한다.

    • 2.file 항목에 hdfs브라우저에서 basicavg.py이 저장된 경로를 선택한다.

    • 그 외 항목을 아래와 같이 입력한다.

      propertyvalue
      6.masteryarn
      7.modeclient
      9.jarbasicavg.py

    • 10.sparkOpts : 위 매뉴얼 중 property 항목을 참고하여 입력한다. 본 예제에서는 driver-memory, conf 옵션을 사용했다.

      keyvalue
      --driver-memory2G
      --confspark.executor.extraJavaOptions=-Xms512m -Xmx1024m
