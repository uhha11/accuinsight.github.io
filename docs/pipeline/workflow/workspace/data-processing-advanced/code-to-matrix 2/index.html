codeToMatrix

Definition#

컬럼을 지정하여 해당 컬럼의 모든 코드값을 신규 컬럼으로 추가하고 매핑되는 컬럼의 값은 1, 매핑되지 않으면 0으로 생성합니다. 예를 들어 A라는 컬럼의 코드 분포가 01,02,03이면 C_01,C_02,C_03이란 신규컬럼이 생성되며, A의 값이 01인 행은 컬럼 C_01의 값이 1이고 C_02, C_03의 값은 0이 됩니다. 좌측 [데이터처리(고급)]노드 중 [codeToMatrix]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우 생성] > [설정]을 참고합니다.

property#

[Node Description] 작성 중인 노드명 입력

  1. targetColumn : 목록 중 기준컬럼을 선택
  2. codeList : 코드 목록 정의

Example#

코드값으로 구성된 타겟컬럼 기준으로 코드값 별 신규칼럼 생성 및 매트릭 구성하는 워크플로우를 생성합니다.

  1. [HDFS불러오기], [codeToMatrix] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [codeToMatrix]노드에 아래와 같이 입력 후 snapshot 클릭
