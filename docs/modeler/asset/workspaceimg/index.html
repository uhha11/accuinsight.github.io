워크스페이스 이미지

워크스페이스 이미지 관리#

  1. 워크스페이스 이미지 목록
    • 워크스페이스 이미지 목록이 리스트형태로 화면에 나타납니다.
  2. 검색기능
    • 워크스페이스 이미지 목록 셀에 있는 텍스트를 기반으로 검색해서 필터링 해줍니다.
  3. 편집
    • 해당 이미지에 대한 이름, 설명 및 공유범위를 지정합니다.
  4. 삭제
    • 해당 이미지를 삭제합니다.
  5. Requirements
    • 해당 이미지가 실행될 때 필요한 라이브러리 및 툴의 버전 정보가 화면에 나타납니다.

워크스페이스 공유 이미지#

워크스페이스 이미지 목록에서 공유된 이미지만 필터링해서 이미지의 프로젝트명과 함께 화면에 나타납니다.

  1. Requirements
    • 해당 이미지가 실행될 때 필요한 라이브러리 및 툴의 버전 정보가 화면에 나타납니다.
