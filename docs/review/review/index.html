#DL Modeler

  1. DL Modeler Programming Guide 미작성. 작성여부 검토 필요
  2. 프로젝트 생성/수정시 CPU/GPU 인스턴스 개수 선택/수정 항목
  • 클라우드 버전의 경우 자원제약이 없어 프로젝트 생성시 GPU/CPU 선택 없음
    • custom 작업 선택시 cpu/gpu 선택
  • 온프레미스 버전의 경우 가용자원이 유한하므로 아래내용 필요함(프로젝트 챕터에도 반영)

Note On-Premise 환경의 경우 가용자원이 유한하므로 프로젝트 생성시 인스턴스를 선택한다.
프로젝트 생성/수정시 GPU/CPU 인스턴스 및 개수를 선택할 수 있다.

  1. 프로젝트 생성시
  • 프로젝트 자원선택의 [GPU 개수]는 프로젝트에서 사용할 수 있는 전체 GPU 개수를, [인스턴스 자원]은 프로젝트에서 사용할 수 있는 전체 인스턴스 개수(GPU인스턴스+CPU인스턴스)를 의미한다.
  • 예를 들어, GPU 개수를 2, 인스턴스 자원을 5로 입력할 경우 수행가능한 작업은 다음과 같다.
    • GPU 2개짜리 학습을 하나 돌릴 경우, cpu를 사용하는 인스턴스는 4개를 돌릴 수 있음
    • GPU 1개짜리 학습을 하나씩 2개 돌릴 경우, cpu를 사용하는 인스턴스는 3개를 돌릴 수 있음
  • 프로젝트 생성 창의 GPU 개수는 프로젝트에서 사용할 수 있는 GPU의 최대개수를 지정(limit 설정)하는 의미이고, CPU 인스턴스의 경우 인스턴스당 한개의 CPU를 사용할 수 있다.
  1. 프로젝트 수정시
  • 프로젝트 수정 화면에서 프로젝트에 할당한CPU/GPU 자원의 양을 확인할 수 있다.
