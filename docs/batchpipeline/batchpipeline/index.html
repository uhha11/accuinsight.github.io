Batch Pipeline

Batch Pipeline#

1. Batch Pipeline 개요#

1.1. Batch Pipeline 소개 및 주요기능#

Batch Pipeline은 on-Premise 데이터뿐만 아니라, Cloud 컴퓨팅, 스토리지 서비스 간 대규모 데이터를 안정적으로 처리하고 Job 스케줄링과 모니터링 기능을 제공하는 솔루션입니다. UI/UX 기반의 다양한 데이터 편집기능을 Drag & Drop하여 손쉽게 Workflow를 설계할 수 있고, 이렇게 설계된 Batch Job을 실행/제어할 수 있습니다. 또한, Workflow 실행상태 및 실행이력 등을 바로 확인할 수 있도록 직관적인 DashBoard를 제공합니다.

Batch Pipeline에서 제공하는 기능은 다음과 같습니다.

(1) 컴포넌트 제공 및 스케줄 관리 : 다양한 하둡 에코 작업 / 시스템 작업을 컴포넌트 형태로 제공하여 쉽게 설정이 가능합니다. 또한, 시작 시각과 종료 시각, 스케줄 주기 설정이 용이합니다.

(2) 데이터 조회 및 분석 작업 지원 : 워크플로우를 작성하면서 인터렉티브하게 데이터를 확인 할 수 있습니다.

(3) 변수 처리 지원 : 일반/날짜 파라미터, 데이터 셋 등 다양한 동적 변수를 지정 가능합니다.

(4) 모니터링 제공 : 배치 잡(Job) 별 상태 및 실행 정보를 직관적인 UI로 확인 할 수 있습니다.

(5) 다양한 관리 도구 : HDFS, Hive, S3 등의 다양한 브라우저와 관리 기능을 제공합니다.

1.2. 참고#

Batch Pipeline은 워크플로 및 스케줄 처리 엔진으로 Apache Oozie(Workflow Scheduler for Hadoop)를 사용하였으므로 Apache Oozie에 대한 기본 지식이 있으면 매뉴얼을 이해하는데 많은 도움이 됩니다.

Apache Oozie Document는 아래의 경로를 참고 하시기 바랍니다.

http://oozie.apache.org/

http://oozie.apache.org/docs/4.2.0/index.html
