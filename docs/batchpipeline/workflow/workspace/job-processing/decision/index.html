decision

#decision

조건에 따라 작업 flow를 분기시켜 노드들을 독립된 flow에서 수행한다. 분기된 작업은 최종적으로 join으로 합쳐줘야 한다. 좌측 [Flow구성]노드 중 [correlation]노드를 drag & drop 한 후 Property 항목을 입력한다.

  1. switch : 조건문과 이동할 action을 입력한다.
    • case : 조건문
    • to : 이동할 action
  2. default : 기본으로 이동할 action
    • to : 노드 선택
