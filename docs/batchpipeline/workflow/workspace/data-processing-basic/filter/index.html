filter

#filter

원하는 데이터만 불러오기 위해 필터링 조건을 설정한다. 좌측 [데이터처리(기본)]노드 중 [filter]노드를 drag & drop 한다. Property 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 Property 항목을 볼 수 있다.

  1. filters : filtering 옵션을 선택한다.

    • col : 필터링 대상 컬럼 선택한다. 확인해야 할 컬럼이 여러 개일 경우 좌측 [+]버튼을 선택한다.
    filterOption
    less than(<)
    less than or equal(<=)
    equal(=)
    not equal(!=)
    greater than or equal(>=)
    greater than(>)
    contains
    not contains
    like
    • filterValue : 필터 기준값을 입력한다. 문자열 입력시 쌍따옴표("")를 사용하여 문자열을 감싸줘야 한다.

  2. filterOperator : 필터 사이의 조건 연산자를 선택(AND, OR)한다.

  3. useCustom : 사용자 입력 기능 사용 시 선택한다.

  4. customCondition : 사용자가 원하는 조건식을 직접 입력한다. 1번 Property로 해결할 수 없는 복잡한 조건의 경우 사용한다.
