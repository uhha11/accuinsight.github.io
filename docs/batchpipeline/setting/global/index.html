변수관리#

변수관리 개요#

워크플로우 실행 시 사용하는 글로벌 변수를 관리한다. 생성한 변수 그룹은 워크플로우 생성 화면의 parameter 탭 globalVariable에서 선택하여 사용할 수 있으며 parameters에 선언된 변수보다 우선순위가 높다.

① 검색 : 검색할 변수를 입력하여 검색

② 추가 : 변수를 추가한다.

③ 삭제 : 삭제할 변수를 목록에서 체크 후 삭제 버튼을 클릭하여 삭제

④ 변수 목록을 조회한다.

A. 그룹 명 : 변수를 구분하기 위한 이름

B. 변수 목록 : 변수 그룹에서 사용하는 변수 목록

C. 변수 개수 : 변수 그룹에서 사용하는 변수 개수

D. 반복 횟수 : 변수의 값을 다르게 하여 워크플로우 잡을 수행할 횟수

⑤ 편집 : 클릭하여 변수 정보 편집

⑥ 삭제 : 클릭하여 해당 변수 삭제

변수 편집기#

변수 정보를 생성/수정한다.

① 그룹 명 : 변수 정보를 구분하기 위한 변수 그룹 명

② 변수 추가 : 사용할 변수를 추가한다.

③ 반복 추가 : 반복 횟수를 추가 하며 변수의 값을 다르게 설정하여 잡을 수행할 수 있다.

④ csv 불러오기 : csv 파일로 작성된 변수 정보를 로드 할 수 있다. 추가되는 변수 값의 행 수만큼 반복 횟수가 추가되며 use_yn은 Y로 셋팅된다.

⑤ csv 내보내기 : 변수 정보를 csv 파일로 다운로드 할 수 있다. 파일에는 변수 명과 변수 값이 포함된다.

⑥ 저장 : 변수 정보를 저장한다.

⑦ 반복 횟수를 추가하는 경우 행이 추가 되며 해당 반복의 수행여부를 use_yn 컬럼의 Y/N 값으로 설정하여 사용할 수 있다. 반복 횟수만큼 잡에 수행된다.

⑧ 변수를 추가하는 경우 추가하는 변수 명으로 컬럼이 추가 되며 워크플로우에서 필요한 변수 개수만큼 추가하여 사용할 수 있다.
