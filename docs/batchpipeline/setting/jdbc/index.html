#JDBC 관리

#JDBC관리 개요

JDBC 접속 정보를 관리한다. 워크플로우 생성/수정 시 RDB 가져오기, RDB내보내기, query 노드에서 JDBC 접속 정보를 로드하여 사용할 수 있다.

① 검색 : 검색할 JDBC 접속 정보를 입력하여 검색

② 추가 : JDBC 접속 정보를 추가한다.

③ 삭제 : 삭제할 JDBC 접속 정보를 목록에서 체크 후 삭제 버튼을 클릭하여 삭제

④ JDBC 접속 정보 목록을 조회한다.

⑤ 편집 : 클릭하여 JDBC 접속 정보 편집

⑥ 삭제 : 클릭하여 해당 JDBC 접속 정보 삭제

#JDBC 추가/편집

JDBC 접속 정보의 추가/편집버튼 클릭 시 팝업 화면에서 등록/수정/초기화를 할 수 있다.

A. 이름 : JDBC 접속 정보를 구분하기 위한 이름 B. Driver : JDBC 접속을 위한 driver class C. URL : DB 접속 url D. User Name : DB 유저 ID E. Password : DB 유저 패스워드 F. Table Name : 테이블명
